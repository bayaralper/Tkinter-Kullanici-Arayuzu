Projenin amacı:Bu proje github repomda da bulunan Haber Sitesi Scrape adlı projenin kullanıcı arayüz kısmıdır.
Kullanıcının isteğine göre verilerin şekillenmesini ve filtrelenmesini sağlamaktadır.
Bu projenin çalışması için Web scraping projesinin daha önce çalıştırılması veya istenilen dizin/veri tabanında
verilerin bulunması gerekmektedir.


Proje firebase database ile bağlanmalıdır.
